# Bank Account Management System

A full-stack Bank Account Management System built using Angular (frontend) and Java Spring Boot (backend). The application allows users to manage bank accounts securely with features like account creation, login, deposits, withdrawals, and transaction history.

## 🚀 Technologies Used

### Frontend
- Angular
- HTML5
- CSS3
- Bootstrap
- TypeScript

### Backend
- Java
- Spring Boot
- Spring Data JPA
- REST APIs

### Database
- MySQL

## ✨ Features

### User Features
- User Login System
- Secure Authentication
- Create Bank Account
- View Account Details
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History

### System Features
- RESTful API Integration
- MVC Architecture
- Database Persistence using JPA/Hibernate
- Responsive UI
- Modular Code Structure
